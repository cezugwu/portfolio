Ecommerce Website (EMK-Xpress)

Overview

This is a fully functional eCommerce platform built using Django (backend) and React (frontend). It provides a seamless shopping experience with a modern UI, secure authentication, and smooth payment integration.

Tech Stack

Backend – Django, Django REST Framework (DRF), PostgreSQL
Frontend – React, Tailwind CSS, Redux (if used)
Authentication – JWT Authentication
Payment Integration – Flutterwave & Paystack
Hosting – (Mention deployment platform, e.g., Vercel, AWS)

Key Features

User Authentication – Secure login and registration using JWT
Product Management – Admin can add, update, and delete products
Cart & Checkout – Users can add items to the cart and make purchases
Payment Gateway – Secure transactions via Flutterwave
Order History – Users can track past purchases
Search & Filtering – Find products quickly with search and filters
Responsive Design – Works on all devices

Challenges & Solutions

Handling Payment Processing – Integrated Flutterwave API with error handling
Optimizing API Performance – Used Django caching and query optimizations
Ensuring Security – Implemented JWT authentication and CSRF protection

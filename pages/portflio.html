Portfolio (A Modern & Interactive Portfolio Landing Page)

Overview

Portfolio is a sleek and responsive personal portfolio landing page designed to showcase skills, projects, and achievements in an engaging and visually appealing manner. Built with modern web technologies, it features smooth animations, an intuitive user experience, and a dark mode toggle for enhanced accessibility.

Tech Stack

Frontend – Tailwind CSS, Html and JavaScript
Hosting – AWS

Key Features

Responsive Design – Ensures a seamless user experience across all devices.
Smooth Animations & Transitions – Integrated Framer Motion for fluid interactions.
Dark Mode Integration – Users can toggle between light and dark themes.
Intuitive Navigation –Well-structured layout with easy access to different sections.

Challenges & Solutions

Performance Optimization – Minimized animations for better load speed without sacrificing UI quality.
Dark Mode Implementation – Used Tailwind’s dark mode support for effortless theme switching.
Ensuring Accessibility – Applied proper color contrasts and keyboard navigation for usability.
